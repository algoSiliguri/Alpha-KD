## What P1.5 Added

### 1. `get_state()` — Universal Introspection Hook

Every strategy now answers the question: *"What are you doing right now?"*

`alpha_kd/strategies/base.py` gained a concrete `get_state()` method on the `Strategy` ABC. Call it on any strategy instance and it returns a plain Python dict:

```python
{
  "side": "buy",           # or "sell" or "flat"
  "entry_price": 1.0823,   # None if flat
  "unrealized_pnl": None,  # populated by concrete subclasses
  "signal": 0,             # 0 by default; overridden by subclasses
  "timestamp": None        # overridden by subclasses
}
```

All values are JSON-serializable. Subclasses override only what they know.

### 2. `RsiSma.get_state()` — Concrete Override

`RsiSma` now overrides `get_state()` to populate `unrealized_pnl` from the live position and `timestamp` from the last bar in `self.data`. When long, unrealized P&L = `(last_close - entry) / entry`. When short, it's the inverse. Signal is set to `1` or `-1` to match the position direction.

### 3. `TelemetryBuffer` — Append-Only JSONL Ring Buffer

`alpha_kd/telemetry.py` implements a write-friendly log that:
- Appends one JSON line per `record()` call (no in-memory buffering)
- Automatically truncates to the last `max_records` (default 10,000) when exceeded
- Uses a write-then-rename pattern for safe file replacement
- Has zero third-party dependencies (stdlib `json`, `pathlib` only)

### 4. `RegimeMixin` — Opt-In Regime Bridge

`alpha_kd/strategies/regime_mixin.py` lets any `Strategy` subclass inherit regime awareness:

```python
class MyRegimeStrategy(RegimeMixin, Strategy):
    ...
```

If `statsmodels` or the `regime_detection` package is unavailable, `current_regime` silently becomes `"unavailable"` and the strategy works normally. No crash, no import error.

### 5. CLI State Dump

```bash
python -m alpha_kd --state
```

Prints the most recent telemetry record as formatted JSON, or `"No telemetry records found."` on an empty buffer. Accepts `--telemetry-path` to point at a custom JSONL file.

# Smoke-test TelemetryBuffer ring-buffer truncation
python -c "
import tempfile, json
from pathlib import Path
from alpha_kd.telemetry import TelemetryBuffer
with tempfile.TemporaryDirectory() as tmp:
    buf = TelemetryBuffer(Path(tmp) / 'tel.jsonl', max_records=10_000)
    for i in range(12_000):
        buf.record({'i': i})
    lines = (Path(tmp) / 'tel.jsonl').read_text().splitlines()
    print(f'Lines after 12k records: {len(lines)} (expected 10000)')
    assert len(lines) == 10_000
"
